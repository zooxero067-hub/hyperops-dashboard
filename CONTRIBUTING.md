# HyperOps 대시보드 — 월별 데이터 추가 가이드

> **이 작업은 코딩 지식이 없어도 됩니다.**  
> CSV 파일만 준비하면 Claude가 모든 기술 작업을 처리합니다.

---

## 전체 흐름

```
① 분석 플랫폼에서 CSV 다운로드
        ↓
② 클로드 CLI 열기 (프로젝트 폴더에서)
        ↓
③ 아래 프롬프트 복붙 + CSV 파일 경로 입력
        ↓
④ 클로드가 알아서 처리 → 자동 배포
```

---

## Step 1. CSV 파일 준비

자사몰 분석 플랫폼에서 **해당 월 1일~말일** 기간으로 아래 리포트를 CSV로 내려받습니다.

| 리포트 이름 | 필수 여부 |
|------------|----------|
| 매출 종합 분석 (일별) | 필수 |
| 전체 방문자 수 (일별) | 필수 |
| 구매단계 분석 | 필수 |
| 연령별 매출 분석 | 필수 |
| 지역별 매출 분석 | 필수 |
| 재구매까지 걸린 시간 | 필수 |
| 유입도메인 (상세) | 필수 |
| 검색엔진 (상세) | 필수 |
| 처음 방문 / 재방문 구매 | 필수 |
| 상품별 매출 분석 | 필수 |
| 시간대별 평균 접속 수 | 없으면 생략 가능 |

다운받은 CSV 파일을 한 폴더에 모아둡니다.  
예) `/Users/본인이름/Downloads/2026년4월/`

---

## Step 2. Claude CLI 열기

터미널(Mac: `command + 스페이스` → "터미널" 검색)을 열고 아래 명령어를 입력합니다.

```
cd ~/Developer/02-hyperops
claude
```
