# HyperOps 대시보드 — 월별 데이터 추가 가이드

> **이 작업은 코딩 지식이 없어도 됩니다.**  
> CSV 파일만 준비하면 Claude가 모든 기술 작업을 처리합니다.

---

## 전체 흐름

```
① 자사몰 + 스마트스토어 CSV 다운로드
        ↓
② 클로드 CLI 열기 (프로젝트 폴더에서)
        ↓
③ 아래 프롬프트 복붙 + CSV 파일 경로 입력
        ↓
④ 클로드가 알아서 처리 → 자동 배포
```

---

## Step 1. CSV 파일 준비

### 🏠 자사몰 — 에이스카운터에서 다운로드

**기간: 해당 월 1일 ~ 말일**

| 리포트 이름 | 필수 여부 |
|------------|----------|
| 매출 종합 분석 (일별) | 필수 |
| 전체 방문자 수 (일별) | 필수 |
| 구매단계 분석 | 필수 |
| 연령별 매출 분석 | 필수 |
| 지역별 매출 분석 | 필수 |
| 재구매까지 걸린 시간 | 필수 |
| 유입도메인 (상세) | 필수 |
| 검색엔진 (상세) | 필수 |
| 처음 방문 / 재방문 구매 | 필수 |
| 상품별 매출 분석 | 필수 |
| 시간대별 평균 접속 수 | 없으면 생략 가능 |
| 신규회원 일별 현황 | 있으면 추가 |

---

### 🟢 스마트스토어 — 네이버 스마트스토어 판매자센터에서 다운로드

**경로: 판매자센터 → 통계**  
**기간: 해당 월 1일 ~ 말일**

> 파일 형식: 고객현황만 **CSV**, 나머지는 모두 **XLSX**로 다운로드됩니다.

| 리포트 이름 | 위치 | 형식 |
|------------|------|------|
| 고객 현황 (일별 신규/기존 고객 수) | 통계 → 고객분석 | CSV |
| 상품 성과 (상품명, 매출액, 주문 수) | 통계 → 판매분석 → 상품성과 | XLSX |
| 전체 채널 (채널별 방문 수 / 매출) | 통계 → 마케팅분석 → 전체채널 | XLSX |
| 검색 채널 (검색어별 방문 수 / 매출) | 통계 → 마케팅분석 → 검색채널 | XLSX |
| 시간대별 방문 수 | 통계 → 방문분석 → 시간대별 | XLSX |

다운받은 CSV 파일을 한 폴더에 모아둡니다.  
예)
- 자사몰: `/Users/본인이름/Downloads/2026년4월/자사몰/`
- 스마트스토어: `/Users/본인이름/Downloads/2026년4월/스마트스토어/`

---

## Step 2. Claude CLI 열기

터미널(Mac: `command + 스페이스` → "터미널" 검색)을 열고 아래 명령어를 입력합니다.

```
claude
```

## Step 3. 아래 프롬프트 복붙

Claude CLI 창에 아래 내용을 복사해서 붙여넣고, **굵은 부분만 수정**하세요.

```
2026년 **[월]**월 대시보드 데이터 추가해줘.

── 자사몰 (에이스카운터) ──
CSV 폴더 경로: **[자사몰 CSV 폴더 경로]**

파일 목록:
- 매출 종합 분석: **[파일명.csv]**
- 전체 방문자 수: **[파일명.csv]**
- 구매단계 분석: **[파일명.csv]**
- 연령별 매출 분석: **[파일명.csv]**
- 지역별 매출 분석: **[파일명.csv]**
- 재구매까지 걸린 시간: **[파일명.csv]**
- 유입도메인: **[파일명.csv]**
- 검색엔진: **[파일명.csv]**
- 처음방문/재방문: **[파일명.csv]**
- 상품별 매출: **[파일명.csv]**
- 시간대별 접속: **[파일명.csv]** (없으면 이 줄 삭제)
- 신규회원: **[파일명.csv]** (없으면 이 줄 삭제)

── 스마트스토어 (판매자센터) ──
폴더 경로: **[스마트스토어 파일 폴더 경로]**

파일 목록:
- 고객 현황: **[통계_고객현황_....csv]**
- 상품 성과: **[상품성과_....xlsx]**
- 전체 채널: **[전체채널_....xlsx]**
- 검색 채널: **[검색채널_....xlsx]**
- 시간대별: **[시간대별_....xlsx]**

자사몰과 스마트스토어 마케팅 인사이트 모두 10년차 퍼포먼스 마케터 관점으로 작성해줘.
작업 완료 후 GitHub에 푸시까지 해줘.
```
